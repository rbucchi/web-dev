HTMLTHE LANGUAGE FOR BUILDING WEB PAGES

HTML è l’acronimo di HyperText Markup Language e non è un linguaggio di programmazione

È un linguaggio di markup (linguaggio con marcatori), che permette di indicare come disporre gli elementi all’interno di una pagina

Queste indicazioni vengono date attraverso degli appositi marcatori, detti tag, che hanno la caratteristica di essere inclusi tra parentesi angolari 

<!DOCTYPE html>
<html>
<head>

<title>HTML Tutorial</title>

</head>
<body>

<h1>This is a heading</h1>

<p>This is a paragraph.</p>

</body>
</html>

HTMLTHE LANGUAGE FOR BUILDING WEB PAGES

Struttura della pagina

<!DOCTYPE html>
Definisce che si tratta di una pagina secondo lo standard HTML5

<html>
Racchiude tutta la pagina

<head>
Contiene le impostazioni della pagina, ad esempio il titolo

<body>
Contiene tutto il contenuto della pagina 

<!DOCTYPE html>
<meta charset="utf-8">

<head>

<title>HTML Tutorial</title>

</head>
<body>

  <h1>This is a heading</h1>

  <p>This is a paragraph.</p>

</body>
</html>

Tag di apertura, chiusura, contenuto ed elemento.

HTMLTHE LANGUAGE FOR BUILDING WEB PAGES

La prima distinzione tra i tag esistenti è quella tra i Block-level e gli inline level

Scrivere spazi o mandare a capo all'interno del codice html non ha effetti, mentre questa formattazione viene spceficificata appunto dal contenuto (inteso come html) e in un secondo momento dal css.

I block-level incominciano sempre con un newline, prendono tutto lo spazio disponibile in larghezza e hanno un margine.

E.g.: div, form, ol/ul/li, section, table, h1-6

Gli inline elements invece prendono solo lo spazio 'necessario'.

E.g.: span, a, img, br, b, strong, i, button, select,

ID e CLASS

L'attributo id indica un elemento HTML unico all'interno della pagina. L'attributo class invece un'insieme. 
<div class="city">London</div>
.city {
  padding: 10px;

CSS   THE LANGUAGE FOR STYLING WEB PAGES

CSS è l’acronimo di Cascading Style Sheets e designa un linguaggio di stile per i documenti web

Gli stili CSS istruiscono un browser su come il documento debba essere presentato all’utente definendone:
• i font
• i colori
• le immagini di sfondo
• il layout
• etc. 

body {
  background-color: lightblue;
}

h1 {
  color: white;
  text-align: center;
}

p {
  font-family: verdana;
  font-size: 20px;
}

JAVASCRIPT   THE LANGUAGE FOR PROGRAMMING WEB PAGES

Javascript è uno dei linguaggi di programmazione più usati al mondo, anche se con una storia di alti e bassi, la sua consolidata presenza nell’olimpo dei grandi linguaggi come C++, C# e Java è certa

È un linguaggio di programmazione interpretato (non compilato) utilizzato da tutti i browser per permettere la manipolazione dei documenti HTML senza coinvolgere il server. 

<p id=“my">Click the button</p>

<button onclick="myFun()">Click Me!</button>

<script>

  function myFun() {
    var x = document.getElementById("my");
    x.style.fontSize = "25px";
    x.style.color = "red";
  }

</script>
