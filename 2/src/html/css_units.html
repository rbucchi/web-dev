px: Uno dei più usati, è una misura però assoluta. Rimane una misura 'controversa', perché non corrisponde sempre alla misura reale, ma ad una normalizzata per poterla utilizzare in tutti i dispositivi. 
        .px-unit {
            width: 200px;
        }
in, cm, mm, pt (che sono 1/72 di un pollice) e pc (pica che sono 12 pt): Gli svantaggi di px, non utilizzabile su Javascript.. please don't. 
            .in-unit {
                width: 2in;
            }
em, rem, ex: Relativa al font-size. 
                .rem-unit {
                    width: 12em;
                }
vw, vh, vmin e vmax: Relativa al viewport. Viewport è l'area visibile dall'utente. Questo è interessante, sta prendendo piede 'ultimamente' perché inizialmente non era supportato sopprattutto dalle versioni mobile (meh) https://caniuse.com/viewport-units. 
                        .vw-unit {
                            width: 30vw;
                        }
%: Relativa al tag contenitore... Pericolosetto, ma spesso si inizia da qui. 
                        .perc-unit {
                            width: 30%;
                        }
