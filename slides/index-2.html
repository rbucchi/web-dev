HTML   THE LANGUAGE FOR BUILDING WEB PAGES

HTML è l’acronimo di HyperText Markup Language e non è un linguaggio di programmazione

È un linguaggio di markup (linguaggio con marcatori), che permette di indicare come disporre gli elementi all’interno di una pagina

Queste indicazioni vengono date attraverso degli appositi marcatori, detti tag, che hanno la caratteristica di essere inclusi tra parentesi angolari 

<!DOCTYPE html>
<html>

<title>HTML Tutorial</title>

<body>

  <h1>This is a heading</h1>

  <p>This is a paragraph.</p>

</body>
</html>

CSS   THE LANGUAGE FOR STYLING WEB PAGES

CSS è l’acronimo di Cascading Style Sheets e designa un linguaggio di stile per i documenti web

Gli stili CSS istruiscono un browser su come il documento debba essere presentato all’utente definendone:
• i font
• i colori
• le immagini di sfondo
• il layout
• etc. 

body {
  background-color: lightblue;
}

h1 {
  color: white;
  text-align: center;
}

p {
  font-family: verdana;
  font-size: 20px;
}

JAVASCRIPT   THE LANGUAGE FOR PROGRAMMING WEB PAGES

JavaScript è uno dei linguaggi di programmazione più usati al mondo, anche se con una storia di alti e bassi, la sua consolidata presenza nell’olimpo dei grandi linguaggi come C++, C# e Java è certa

È un linguaggio di programmazione interpretato (non compilato) utilizzato da tutti i browser per permettere la manipolazione dei documenti HTML senza coinvolgere il server. 

<p id=“my">Click the button</p>

<button onclick="myFun()">Click Me!</button>

<script>

  function myFun() {
    var x = document.getElementById("my");
    x.style.fontSize = "25px";
    x.style.color = "red";
  }

</script>
