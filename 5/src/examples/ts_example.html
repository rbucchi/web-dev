Typescript example (client)

Il browser non è in grado di interpretare Typescript, per cui bisogna prima compilarlo:


workspace/corso_web (master)$ ./projects/webapi/node_modules/.bin/tsc --sourceMap 5/src/js/ts-example.ts

tsc è il compilatore di Typescript ed è installabile tramite package manager (NuGet o npm ad esempio)


    # -g serve per salvarlo 'globalmente', nella propria macchina di sviluppo
    npm install -g typescript
    # --save-dev serve per salvare da le dipendenze di sviluppo all'interno del package manager
    # verrà salvato nel path locale sotto node_modules
    npm install typescript --save-dev
