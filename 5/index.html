JS Modules

Esempi module: link

Riassunto

I moduli intesi come import/export sono tra le ultime feature implementate di ES6 (caniuse).

Purtroppo dipende tanto dall'env: non esistendo costrutti a tal proposito, ma solamente i 'mezzi'*, ognuno si è implementato il suo metodo e/o filosofia. È probabile che sentiate parlare di CommonJs, AMD, requirejs o ES2015 modules. Quest'ultimi stanno però stabilendo lo standard.

*Per 'mezzi' intendo le primitive js, in questo caso particolare gli scope, chiusure e l'oggetto environment (window). Le varie implementazioni citate (e spesso i ES6 modules) sono variazioni del pattern IIFE o revealing module.

TYPESCRIPTA JAVASCRIPT SUPERSET

Typescript è nato nel 2012 come superset di Javascript il cui scopo è quello di rendere JS più familiare a chi viene da linguaggi OOP e tipizzati.

Non modifica in alcun modo il runtime di JS, in nessun envirnoment. È sempre 'compilato' in JS.

Alcune delle feature introdotte:

  • Static Typing (errori compile time, refences)
  • Classi
  • Interfacce (structural typing e non nominal typing)
  • moduli
  • arrow functions
  • Enumerativi
  • Generici
  • ...
  • async/await
  • optional/default parameters
